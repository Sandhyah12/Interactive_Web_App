# CycleCare - Period Tracker

## Description
CycleCare is a web application that helps users track their menstrual cycle and monitor health information.

## Features
- Track period start date
- Generate cycle calendar
- Record symptoms
- Track water intake

## Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript

##Pages Included

- The web application contains multiple pages:

- Home Page – Introduction and overview of the application.

- Tracker Page – Users can input cycle information and generate predictions.

- History / Calendar Page – Displays the generated cycle calendar.

- Contact Page – Allows users to send feedback or inquiries.

## How to Use
- Open the Home Page of the application.
- Navigate using the navigation bar.
- Go to the Tracker Page and enter:
   - Period start date
   - Cycle length
   - Period length
   - Click Generate Calendar to view predicted cycle days.
   - Record symptoms and water intake if needed.
- Go to History page and see the past cycle history
-Go to Contact page and send the feedback.
